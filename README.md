# WebProgram_JSP

## JSP를 이용한 자료공유 계시판

- 동기 : 게임개발 관련으로 진로를 잡고있고 팀을 구성해 게임을 개발해 보고 싶은 의향이 있습니다. 그래서 팀끼리 나중에 정보를 교환하고 회의하는 사이트를 개설하고 싶어 해당 홈페이지를 제작하기로 함
